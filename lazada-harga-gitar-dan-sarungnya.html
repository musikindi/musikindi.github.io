32. Jika harga 5 potong kain sarung Rp200.000,00, maka harga 3 kodi kain sarung adl​


Jawaban:

Rp2.400.000,00

Penjelasan dengan langkah-langkah:

harga per potong kain sarung adalah Rp200.000,00 / 5 = Rp40.000,00.

Rp40.000,00 x 3 kodi x 20 potong/kodi = Rp2.400.000,00.

Jadi, harga 3 kodi kain sarung adalah Rp2.400.000,00.

Jawaban:

Diketahui harga 5 potong kain sarung adalah Rp200.000,00.

Kita bisa mencari harga per potong dengan cara membagi harga 5 potong dengan 5:

Harga per potong = Rp200.000,00 / 5 potong

Harga per potong = Rp40.000,00/potong

Kodi adalah ukuran jumlah yang setara dengan 20 potong (1 kodi = 20 potong). Oleh karena itu, jika kita ingin mencari harga dari 3 kodi kain sarung, kita perlu mengalikan harga per potong dengan 20, lalu dikalikan dengan 3.

Harga 3 kodi kain sarung = (Harga per potong x 20) x 3 kodi

Harga 3 kodi kain sarung = (Rp40.000,00/potong x 20) x 3

Harga 3 kodi kain sarung = Rp800.000,00

Jadi, harga 3 kodi kain sarung adalah Rp800.000,00.
