26. sebuah silinder homogen bermassa M berjejari R menggelinding tanpa selip turun dari ketinggian h pada sebuah bidang miring dengan sudut kemiringan a. Momentum sudut silinder terhadap sumbu puast massanya setelah mencapai dasar bidang miring adalah a. 4MR √gh/2 b. MR √gh/3 c. 3MR √gh/4 d. MR √gh/5 e. 2MR √gh/6


b. MR√(gh/3)

Pembahasan

Diketahui

Silinder pejal homogen

bermassa Mberjejari Rturun dari ketinggian h pada sebuah bidang miring

Ditanya

Momentum sudut silinder terhadap sumbu pusat massanya setelah mencapai dasar bidang miring.

Proses

Rumus cepat menghitung kecepatan benda yang menggelinding pada suatu bidang miring adalah sebagai berikut:

[tex]\boxed{ \ v = \sqrt{\frac{2gh}{1 + k}} \ }[/tex]

dengan,

v = kecepatan (m/s)

h = ketinggian (m)

k = koefisien inersia benda

g = percepatan gravitasi (m/s²)

Silinder pejal memiliki [tex]\boxed{ \ k = \frac{1}{2} \ }[/tex].

Kita siapkan kecepatan silinder pejal menggelinding turun dari ketinggian h pada sebuah bidang miring.

[tex]\boxed{ \ v = \sqrt{\frac{2gh}{1 + \frac{1}{2}}} \ }[/tex]

[tex]\boxed{ \ v = \sqrt{\frac{2gh}{\frac{2}{3} + \frac{1}{2}}} \ }[/tex]

∴ [tex]\boxed{\boxed{ \ v = \sqrt{\frac{4gh}{3}} \ }}[/tex] ... Persamaan-1

Rumus momentum sudut adalah [tex]\boxed{ \ L = I \omega \ }[/tex], dengan

I = momen inersia, danω = kecepatan sudut

Karena [tex]\boxed{ \ v = \omega R \ }[/tex] dan momen inersia silinder pejal adalah [tex]\boxed{ \ I = \frac{1}{2}MR^2 \ }[/tex]

maka [tex]\boxed{ \ L = \frac{1}{2}MR^2 \Big(\frac{v}{R} \Big) \ }[/tex] ... Persamaan-2

Substitusikan Persamaan-1 ke dalam Persamaan-2.

[tex]\boxed{ \ L = \frac{1}{2}MR^2 \Bigg(\frac{1}{R}\sqrt{\frac{4gh}{3}} \Bigg) \ }[/tex]

[tex]\boxed{ \ L = \frac{1}{2}MR \Bigg(2\sqrt{\frac{gh}{3}} \Bigg) \ }[/tex]

Dengan demikian diperoleh momentum sudut silinder terhadap sumbu pusat massanya setelah mencapai dasar bidang miring, yakni

[tex]\boxed{\boxed{ \ L = MR \sqrt{\frac{gh}{3}} \ }}[/tex]

30. a. nyatakan panjang GH dalam meter b. jiaka panjang GH =20 m, hitunglah panjang AH


a. Panjang GH adalah 10 meter

b. jika panjang GH =20 m, Panjang AH adalah 17,58 meter

Untuk Mencari panjang GH, maka kita menggunakan Teorema Pythagoras

Bunyi Teorema Pythagoras adalah kuadrat sisi miring sama dengan jumlah kuadrat sisi penyikunya

Pembahasan :

Diketahui :

AH = 3m

AB = 1m

BC = 2m

CD = 3m

DE = 4m

EF = 5m

FG = 6m

Ditanya :

a. Panjang GH ?

b. Jika panjang GH = 20m, Maka panjang AH ?

Dijawab :

a. Untuk mencari panjang GH kita gunakan Rumus Pythagoras

AH = 3m

AB = 1m

BH² = AH² + AB²

BH² = 3² + 1²

BH² = 9 + 1

BH² = 10

BH = √10m

BC = 2m

CH² = BH² + BC²

CH² = (√10)² + 2²

CH² = 10 + 4

CH² = 14

CH = √14m

CD = 3m

DH² = CH² + CD²

DH² = (√14)² + 3²

DH² = 14 + 9

DH² = 23

DH = √23m

DE = 4m

EH² = DH² + DE²

EH² = (√23)² + 4²

EH² = 23 + 16

EH² = 39

EH = √39m

EF = 5m

FH² = EH² + EF²

FH² = (√39)² + 5²

FH² = 39 + 25

FH² = 64

FH = 8m

FG = 6m

GH² = FH² + FG²

GH² = 8² + 6²

GH² = 64 + 36

GH² = 100

GH = 10m


b. Untuk mencari Panjang AH kita gunakan Rumus Pythagoras

GH = 20m

FG = 6m

FH² = GH² - FG²

FH² = 20² - 6²

FH² = 400 - 36

FH² = 364

FH = √364m

EF = 5m

EH² = FH² - EF²

EH² = (√364)² - 5²

EH² = 364 - 25

EH² = 339

EH = √339m

DE = 4m

DH² = EH² - DE²

DH² = (√339)² - 4²

DH² = 339 - 16

DH² = 323

DH = √323m

CD = 3m

CH² = DH² - CD²

CH² = (√323)² - 3²

CH² = 323 - 9

CH² = 314

CH = √314m

BC = 2m

BH² = CH² - BC²

BH² = (√314)² - 2²

BH² = 314 - 4

BH² = 310

BH = √310m

AB = 1m

AH² = BH² - AB²

AH² = (√310)² - 1²

AH² = 310 - 1

AH² = 309

AH = √309 = 17,58m
