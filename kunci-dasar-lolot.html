9. sebuah transformator memiliki 30 lililtan primer dan 30 lolotan sekunder.Jika tegangan pda lilitan primer adl 220 volt. tentukan a). tegangan pd lilitan sekunder b). jika arus pd lilitan primer sebesar 0,5 mA berpakah arus listrik yg mengalir pd lilitan sekunder c). efisiensi transformator d). jenis transformator


dik :
Np = 300 lilitan
Ns = 30 Lilitan
Vp = 220 V
jawab :
-------------------------TEGANGAN SEKUNDER--------------
Np / Ns = Vp / Vs
300 / 30 = 220 / Vs
10 = 220 / Vs
Vs = 220 /10
Vs = 22 Volt
------------------------------ARUS SEKUNDER----------------
>> jika arus pada kumparan primer adalah 0,0005 Ampere, maka
Np / Ns = Is / Ip
300 / 30 = Is / 0,0005
10 = Is / 0,0005
Is = 10. 0,0005
Is = 0,005 Ampere
---------------------EFISIENSI TRANSFORMATOR---------------
η = Ps / Pp x 100%
η = ( Vs. Is ) / Vp. Ip ) x 100%
η = ( 22. 0,005 ) / ( 220 . 0,0005 ) x 100%
η = 0,11 / 0,11 x 100%
η = 1 x 100%
η = 100 %
----------------JENIS TRANSFORMATOR-------------
Karna Trafo memiliki kumparan primer lebih banyak dari kumparan sekunder, maka trafo ini termasuk TRANSFORMATOR STEP DOWN
