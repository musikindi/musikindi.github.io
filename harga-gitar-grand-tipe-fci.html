= 8000

Umpamakan
Kamar A = X
Kamar B = Y

Persamaan Matematika
5X + 4Y = 2.500.000 (Persamaan 1)
3X + 2Y = 1.400.000 (Persamaan 2)

5X + 4Y = 2.500.000    * 2
3X + 2Y = 1.400.000    * 4
__________________

10X + 8Y = 5.000.000
12X + 8Y = 5.600.000
-------------------------------- -
-2X = -600.000
X = 300.000

X = 300.000 disubtitusi ke persamaan 1

5X + 4Y = 2.500.000
5(300.000) + 4Y = 2.500.000
1.500.000 + 4Y = 2.500.000
4Y = 2.500.000 - 1.500.000
4Y = 1.000.000
Y = 250.000

Harga 3 Kamar A dan 4 Kamar B
= 3X + 4Y
= 3 (300.000) + 4 (250.000)

39. Sebuah pom bensin menjual tiga tipe bensin. Tipe 1 dijual dengan harga Rp5.000,00/liter, tipe 2 dijual dengan harga Rp6.500,00/liter, dan tipe 3 dijual dengan harga Rp7.500,00/liter. Jika banyak bensin tipe A yang terjual dua kali lebih banyak daripada tipe B, maka banyak bensin tipe B yang terjual pada hari itu adalah​


Jawab:

Banyaknya bensin pertalite dan pertamax yang harus dibeli agar diperoleh keuntungan yang sebesar-besarnya, masing-masing adalah 60 liter dan 40 liter.

Pembahasan

Dimisalkan bensin pertalite adalah x dan bensin pertamax adalah y.

(a) Yang dibeli tidak lebih dari 100 liter bensin pertalite dan pertamax, maka x + y ≤ 100.

(b) Bensin pertalite dibeli dengan harga Rp4 500,00, dan pertamax dengan harga Rp6.500,00 (misal), sedangkan modal hanya Rp530.000,00, maka 4.500x + 6.500y ≤ 530.000 atau bisa disederhanakan menjadi 9x + 13y ≤ 1.060.

(c) Bensin pertalite dijual Rp5.500,00 per liter sehingga keuntungan untuk pertalite Rp5.500,00 – Rp4.500,00 = Rp1.000,00.

Bensin pertamax dijual dengan harga Rp8.000,00 per liter, sehingga keuntungan untuk pertamax Rp8.000,00 – Rp6.500,00 = Rp1.500,00.

Fungsi tujuan (x,y) = 1.000x + 1.500y

Eliminasi pertidaksamaan (a) dan (b) untuk menentukan x:

x + y = 100 |×13| 13x + 13y = 1.300

9x + 13y = 1.060 |×1| 9x + 13y = 1.060 –

4x = 240 ➡ x = 60

Substitusi nilai x ke pertidaksamaan (a) untuk memperoleh nilai y:

x + y = 100

60 + y = 100

y = 100 – 60 ➡ y = 40

Jadi banyaknya bensin pertalite yang dibeli adalah 60 liter dan bensin pertamax sebanyak 40 liter.
