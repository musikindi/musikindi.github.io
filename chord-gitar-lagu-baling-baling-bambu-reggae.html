= 20π rad/s

=20π × 2

36. Baling baling bambu awalnya diam,kemudian berputar dgn kecepatan tetap. Setelah 30 sekon baling baling berputar dgn kelajuan sudut 60 rpm. Tentu kan besar percepatan sudut roda


Diket :
wo = 0 rad/s (awalnya diam)
t = 30 s
wt = 60 rpm = 60 x 2pi/60 = 2pi rad/s

Dit : alfa (percepatan sudut)

Jawab :
wt = wo + alfa.t
2pi = 0 + alfa.30
2pi = 30 alfa
alfa = 2pi / 30
alfa = 0,2 rad/s^2

Jadi, percepatan sudut roda adalah 0,2 rad/s ^2.
